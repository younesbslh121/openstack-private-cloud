## Component Dependencies

| Component | Depends On | Provides |
|-----------|-----------|----------|
| Keystone | MariaDB, Memcached | Authentication tokens, service catalog |
| Nova | Keystone, RabbitMQ, MariaDB, Placement, Neutron, Glance | VM lifecycle management |
| Neutron | Keystone, RabbitMQ, MariaDB, OVS | Virtual networking, floating IPs |
| Glance | Keystone, MariaDB | OS image storage |
| Cinder | Keystone, RabbitMQ, MariaDB, LVM | Block storage volumes |
| Heat | Keystone, RabbitMQ, MariaDB, Nova, Neutron | Stack orchestration |
| Horizon | Keystone, all services | Web dashboard |
| HAProxy | All API services | Load balancing, VIP |
